# Morning Standup Automation Skill v1

## Purpose
Every morning at 7:00 AM MDT, AI posts the daily standup to #leo-auto. Fleet is aligned. Bennett wakes up to a clear picture. No meeting needed.

## Standup Components (compiled automatically)

**1. Overnight Completions** (actions completed since last standup)
- Skills executed, files committed, drive saves, Notion updates
- Any overnight sequences triggered

**2. Today's Active Pipeline**
- Candidates hitting stage milestones today
- Discovery calls scheduled today
- FDD reviews due today
- Awards expected today

**3. Revenue Snapshot**
- Yesterday's QB-verified revenue (from skill-qb-revenue-sync-v1)
- MTD pipeline vs. goal

**4. Active Work Orders**
- Open Leo/Ivan work orders and their status
- Any P1 items requiring same-day completion

**5. Alerts**
- Any red flags from overnight monitoring (Meta Ads, pipeline stalls, dormant leads)
- Any human gate items awaiting Bennett

**6. Today's Top 3 Priorities**
- AI-generated based on pipeline velocity, deal close probability, and time-sensitive items

## Format
```
🌅 FKI MORNING STANDUP — [Date]
[Components 1-6 above]
Ready for the day. Squirrel is live.
```

## Autonomy Value
**Replaces:** Morning team meeting or Bennett manually reviewing all dashboards.
**Result:** Bennett wakes up fully briefed. Day starts with clarity, not catch-up.
**Advaita gap closed:** Daily business visibility — from 0% → 100% AI-autonomous.
