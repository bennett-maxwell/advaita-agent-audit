# New Skill Creation Workflow Skill v1

## Purpose
When a new skill is needed (identified via gap analysis, learning extraction, or ad hoc need), AI follows this workflow to create it correctly. Consistent quality, diamond-verified, immediately usable.

## Skill Creation Process

**Step 1 — Define the Gap**
What specific capability is missing?
What happens today without this skill? (manual work, blocker, or workaround)
What will be possible with this skill?

**Step 2 — Check for Existing Coverage**
SearchKnowledge for similar skills before creating.
If close match exists → extend existing skill rather than create new one.
