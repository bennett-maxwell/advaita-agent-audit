# Prompt Optimization Skill v1

## Purpose
Track which AI-generated outputs Bennett or the team found most useful vs. least useful. Use this feedback to improve prompts and generation quality over time.

## Feedback Collection Triggers
- When Bennett says "that's exactly right" or similar → positive signal
- When Bennett says "no, not like that" → negative signal + correction
- When a generated output was used without modification → positive signal
- When a generated output required significant editing → improvement signal

## Prompt Improvement Log (Drive .learnings/prompts.md)
```
## PROMPT-[YYYYMMDD]-[NNN]
Task type: [what was being generated]
Prompt used: [summary of approach]
Output quality: EXCELLENT / GOOD / NEEDED EDITING / REJECTED
Feedback: [what Bennett said or did]
Improved prompt: [revised approach for next time]
```

## High-Value Prompt Templates (maintain in Drive)
- Discovery call brief format (skill-discovery-call-prep-v1 — most used)
- Candidate scoring narrative
- Pipeline velocity report
- Weekly business review format
- Slack receipt format

## Prompt A/B Framework
When uncertain between two generation approaches → log both, use both in similar situations, compare outcomes, deprecate lower performer.

## Autonomy Value
**Replaces:** AI using same prompts forever regardless of quality.
**Result:** FKI's AI-generated outputs improve every week. Content quality compounds.
**Advaita gap closed:** AI output quality improvement — from 0% → systematic continuous improvement.
