# Autonomy Quarterly Roadmap Skill v1

## Purpose
At the start of each quarter, AI generates a prioritized roadmap for reaching the next Advaita completion milestone. Progress is planned, not accidental.

## Roadmap Generation Process

**Step 1 — Current State Assessment**
Run skill-advaita-completion-score-v1 → get current % by function.

**Step 2 — Gap Prioritization**
From skill-gap-analysis-execution-v1 → get current P1/P2/P3 gap list.
Sort by: impact on Bennett's time × feasibility × time to build.

**Step 3 — Quarterly Target Setting**
Given current completion and gap list → what is achievable in 90 days?
Propose specific target: "Q3 2026: Reach 75% overall Advaita completion by [date]"

**Step 4 — Skill Roadmap**
List: top 10 skills to build this quarter (from gap list) in priority order.
For each: estimated time to build, pillar, expected completion % increase.

**Step 5 — Dependency Mapping**
Are any roadmap skills dependent on others? Build dependency order.

## Quarterly Roadmap Format (posted to #leo-auto at quarter start)
```
🗺️ ADVAITA ROADMAP — Q[N] [YEAR]
Current completion: X%
Quarter target: X%
Top 10 skills to build:
1. [Skill name] — [pillar] — [expected impact]
...
Estimated completion by end of quarter: X%
On track to reach 100% by: [projected date]
```

## Autonomy Value
**Replaces:** Advaita progress happening randomly based on what Bennett notices.
**Result:** Systematic quarterly planning ensures FKI reaches full AI autonomy on a predictable timeline.
**Advaita gap closed:** Strategic autonomy planning — from 0% → 100% systematic.
