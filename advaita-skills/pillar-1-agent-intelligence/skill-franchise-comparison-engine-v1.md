# Franchise Comparison Engine Skill v1

## Purpose
When a candidate is deciding between 2–3 brands on their shortlist, AI generates a structured side-by-side comparison anchored to their specific priorities.

## Comparison Dimensions (pulled from FDD + brand data)

| Dimension | Brand A | Brand B | Brand C |
|-----------|---------|---------|---------|
| Investment range (per FDD Item 7; results may vary) | | | |
| Royalty + marketing fee (FDD Item 6) | | | |
| Territory size and exclusivity (FDD Item 12) | | | |
| Semi-absentee compatible? | | | |
| Financial performance rep (FDD Item 19 disclosed?) | | | |
| Number of open units (FDD Item 20) | | | |
| Franchisor years in franchising (FDD Item 1) | | | |
| State availability for candidate's market | | | |
| Culture fit score (from brand-fit-matching-v1) | | | |

## Personalized Priority Weighting
AI asks candidate: "What are the 3 most important factors for you?" → Weights comparison accordingly.
- If capital-first: Investment range and fee structure emphasized
- If lifestyle-first: Semi-absentee compatibility emphasized
- If ROI-first: Item 19 data emphasized (results may vary; see FDD)

## Recommendation Synthesis
After comparison: AI generates 1-paragraph recommendation based on weighted priorities.
"Based on your priorities of [X, Y, Z], [Brand] is the strongest fit because [specific reasons]."

## Autonomy Value
**Replaces:** Broker building comparisons ad hoc, often biased toward easiest-to-explain brand.
**Result:** Data-driven, candidate-priority-weighted comparison. Confident decision-making.
**Advaita gap closed:** Brand comparison intelligence — from 0% → 95% AI-autonomous.
