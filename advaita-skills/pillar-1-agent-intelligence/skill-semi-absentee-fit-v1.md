# Semi-Absentee Fit Assessment Skill v1

## Purpose
Determine if a candidate is genuinely suited for semi-absentee ownership. Mismatches cause post-award frustration. This skill prevents the mismatch.

## Semi-Absentee Suitability Criteria

**MUST HAVE (all required)**:
- [ ] Willing to hire and trust a general manager
- [ ] Comfortable with remote oversight (weekly check-ins vs. daily presence)
- [ ] Has or is willing to develop management/delegation skills
- [ ] Financial cushion to support initial period before unit reaches target cash flow
- [ ] Timeline allows 3–12 month ramp before reaching semi-absentee state

**NICE TO HAVE (strengthens fit)**:
- Prior experience managing people (even in corporate context)
- Existing community connections in target market (especially IC)
- Complementary W2 income during ramp period
- Clear "why" for not being hands-on (second career, current job, other investments)

**DISQUALIFYING SIGNALS**:
- Needs immediate income from Day 1 (semi-absentee rarely cash-flows immediately)
- Uncomfortable with uncertainty or delegation
- "I'll just check in occasionally" (underestimating commitment required in Year 1)
- Partner/investor situation with unclear decision authority

## Output
QUALIFIED for semi-absentee: All MUST HAVEs met
CONDITIONAL: 1 MUST HAVE missing — specific development action required
NOT RECOMMENDED: 2+ MUST HAVEs missing — owner-operator path instead

## Autonomy Value
**Replaces:** Broker making semi-absentee recommendation without structured assessment.
**Result:** Only truly suitable candidates get IC/SRP recommendation. Post-award satisfaction increases.
**Advaita gap closed:** Semi-absentee qualification gate — from 0% → 95% AI-autonomous.
