# Reversible Action Gate Skill v1

## Purpose
Before executing ANY action, AI classifies it as REVERSIBLE or IRREVERSIBLE and routes accordingly. The foundation of autonomous execution without risk.

## REVERSIBLE (execute immediately, post receipt)
- Notion creates/updates
- Google Drive writes
- GitHub commits
- Slack posts
- GHL tag changes
- GHL stage changes
- GHL note additions
- GHL sequence enrollment (not financial)
- Sprint Board updates
- Memory writes
- Skill file updates
- Dashboard data refreshes
- Email drafts (not sends)
- Report generation
- Data reads and analysis

## IRREVERSIBLE (daily digest ONLY — never auto-execute)
- Outbound emails TO external parties (candidates, franchisors, partners)
- Meta Ads budget changes
- Meta Ads campaign start/pause/delete
- GHL contact deletion or hard archive
- External contract or agreement signatures
- Any financial transaction
- Public website content changes (once indexed, permanent SEO impact)
- Franchisor communication
- Any action that cannot be undone in <5 minutes

## Ambiguous Actions → Always classify IRREVERSIBLE
When in doubt: IRREVERSIBLE. Log to digest. Let Bennett decide.

## Classification Logging
Every action logged with classification before execution:
`[TIMESTAMP] ACTION: [description] CLASSIFICATION: REVERSIBLE/IRREVERSIBLE AUTHORITY: SQUIRREL/DIGEST/BENNETT`

## Autonomy Value
**Replaces:** Agent guessing what's safe to execute.
**Result:** Zero unauthorized permanent actions. Bennett's trust in AI autonomy grows continuously because AI never overreaches.
**Advaita gap closed:** Autonomous execution authority — from 0% → 100% properly classified.
